#1. 반가워요!
- 이걸 배포하는 이유는, 팀 포트리스 2가 동아시아권 번역안을 더 이상 받지 않아서 "반글화" 상태에 놓였기 때문입니다.
- 특별한 업데이트가 없어서 간간히 업데이트되는 것만 받아서 사용하시면 됩니다.

- Github
https://github.com/M60TM/TF2-Korean-Translation-Reborn

#2. 설치 방법
- 이 글을 읽고 있다면, 제 Github에서 압축 파일을 다운받으셨을 겁니다. 그러면, 다음과 같은 과정을 거쳐주세요.
1) 설치한 파일의 압축을 풀어줍니다.
2) 스팀 라이브러리에서 Team Fortress 2를 우클릭하면 속성이 나옵니다. 이것을 클릭해주세요.
3) 속성을 누르시면 로컬 파일 부분에 가서 찾아보기 버튼을 눌러주세요.
4) 찾아보기 버튼을 누르시면, 팀 포트리스 2를 다운받은 폴더로 이동하게 되는데, 여기서 tf 폴더를 눌러주세요.
5) tf 폴더 안에 custom 폴더로 들어가세요.
6) custom 폴더 안에 압축을 풀어놓았던 파일 중에서 "tf2_korean"이라고 이름 붙여진 폴더 또는 tf2_korean.vpk 파일 중 하나를 넣어주세요.
7) 이제 됐습니다! 팀 포트리스 2를 실행하면 됩니다!
